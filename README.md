# DwaniMate

DwaniMate is a bilingual AI-powered voice assistant designed to help users in Karnataka access government services, transport routes, and public schemes using conversational AI in Kannada and English.

## Features
- Bilingual chat interface (Kannada and English)
- Government schemes information
- Smart transport route planning
- AI voice responses
- Real-time location support

## Tech Stack
Frontend
- React
- TypeScript
- Tailwind CSS

Backend
- Python Flask
- Dwani API

## How to Run

Frontend
npm install  
npm run dev  

Backend
pip install -r requirements.txt  
python app.py
